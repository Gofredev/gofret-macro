**Gofret Macro** is a specialized, lightweight **auto-clicking utility** designed specifically for **Blade Ball Clash**. It focuses on providing consistent click speeds and reducing input latency for a better gameplay experience.

---

## 🛠 Features

* 🚀 **Low Latency:** Engineered for the fastest possible response time.
* ⚙️ **Pure Automation:** A clean, efficient auto-clicking script.
* 💎 **User-Friendly:** Simple interface—no complex setup required.
* 🌀 **Consistent Performance:** Helps maintain steady timing during intense clashes.

---

## 📥 How to Install

1.  Go to the [Releases](https://github.com/gofredev/gofret-macro/releases) section.
2.  Download the latest `GofretMacro.exe`.
3.  Double-click the application to launch.
4.  Activate it within your game and experience the consistency of **Gofret**.

## 📜 Information

**Gofret Macro** is an automation tool designed for personal use and accessibility purposes. It does not modify game files or inject code into the game client. Please use it in accordance with the game's fair play guidelines.
